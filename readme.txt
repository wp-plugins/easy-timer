=== Membership Manager ===Contributors: KleorTags: area, areas, member, members, membership, managerRequires at least: 3.2.1Tested up to: 3.4.2Stable tag: 4.7Allows you to create and manage your members areas.== Description ==Membership Manager allows you to create and manage your members areas.- [Documentation](http://www.kleor-editions.com/membership-manager/documentation)== Installation ==1. Unzip the plugin file.2. Upload the "membership-manager" folder to your plugins directory.3. Activate the plugin through the "Plugins" menu in WordPress.4. Personalize the plugin interface through the Membership Manager back office page in WordPress.5. Configure the plugin through the Membership Manager options page in WordPress.== Upgrade Notice ==1. Unzip the plugin file.2. Upload the "membership-manager" folder to your plugins directory.3. Personalize the plugin interface through the Membership Manager back office page in WordPress.4. Configure the plugin through the Membership Manager options page in WordPress.Normally, you don't need to deactivate and reactivate the plugin. However, if you encounter problems after the update, deactivate and reactivate the plugin through the "Plugins" menu in WordPress.== Changelog ==* Version 4.7	* Add the ability to use CAPTCHAs in
